# Hello World

欢迎来到我的博客！这是我的第一篇文章，使用 Hexo 和 Butterfly 主题创建。

## 关于这个博客

这个博客使用了 [Hexo](https://hexo.io/) 静态博客生成器和 [Butterfly](https://butterfly.js.org/) 主题。Butterfly 是一个美观且功能丰富的 Hexo 主题，提供了许多现代化的功能和设计。

## 主要特性

- 🦋 **美观的界面**：Butterfly 主题提供了简洁优雅的设计
- 🌙 **暗色模式**：支持明暗主题切换
- 📱 **响应式设计**：在各种设备上都能良好显示
- 🏷️ **标签和分类**：支持文章分类和标签管理
- 💬 **评论系统**：可集成多种评论服务
- 🔍 **搜索功能**：支持本地搜索和 Algolia 搜索

## 代码高亮

Butterfly 主题支持代码高亮，下面是一个简单的示例：

```javascript
function sayHello() {
  console.log("Hello, World!");
  return "Welcome to my blog!";
}

sayHello();
```

## 数学公式

主题还支持数学公式渲染（需要配置 MathJax 或 KaTeX）：

当 $a \ne 0$ 时，方程 $ax^2 + bx + c = 0$ 的解为：

$$ x = {-b \pm \sqrt{b^2-4ac} \over 2a} $$
